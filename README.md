# Silaii - Fashion Store

## Project Overview
Silaii is an attractive and responsive fashion e-commerce web application. It provides a seamless shopping experience with a modern design, allowing users to browse through various categories, view featured products, and manage their shopping carts. A dedicated admin panel is included for managing products and users, making it a comprehensive solution for online fashion retail.

## Installation
To get started with the Silaii project, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone [repository-url]
   cd silaii-fashion-store
   ```

2. **Open the index.html file in your preferred web browser.**
   You can do this by double-clicking on the file or using a local server if needed.

## Usage
Open the index.html file in a web browser to view the application. Navigate through different sections:
- **Home:** Displays the hero section with a call-to-action button to shop.
- **Shop by Category:** Browse clothing, shoes, accessories, and bags.
- **Featured Products:** View an array of featured products with details.
- **Admin Dashboard:** Accessible via the admin.html file, it allows administrators to manage products and users.

## Features
- Responsive design that adjusts for both desktop and mobile devices.
- User-friendly interface with clear navigation.
- Search functionality for easy product discovery.
- Admin panel for managing users and products.
- Product categories for organized browsing.
- Featured products section to highlight new arrivals and sales.

## Dependencies
No additional dependencies are listed in package.json since the project primarily relies on CDN links, including:
- [Tailwind CSS](https://tailwindcss.com/)
- [Font Awesome](https://fontawesome.com/)

## Project Structure
The project consists of the following files:

```plaintext
.
├── index.html          # Main landing page of the fashion store
└── admin.html          # Admin dashboard for managing products and users
```

### index.html 
- Contains sections for navigation, hero display, product categories, featured products, and a footer.
- Includes a login modal for customer and admin sign-ins.

### admin.html 
- Displays the admin dashboard with options to manage products, users, and analytics.
- Shows a logged-in admin's email and logout functionality.
